# Hasaki Research Playbook

Use this playbook for source-sensitive or external-fact tasks.

## Defaults
- Inspect local wiki/raw sources first.
- Browse only for unstable or external facts.
- Keep concrete dates in findings.
- Include source links for recommendations.

## Suggested Flow
1. Read local files in allowed knowledge scope.
2. Validate assumptions against approved specs and open questions.
3. If needed, browse official/primary sources.
4. Summarize with file paths, requirement refs, and impact notes.
